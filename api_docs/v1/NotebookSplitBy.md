# NotebookSplitBy

Object describing how to split the graph to display multiple visualizations per request.

## Properties

| Name     | Type                   | Description       | Notes |
| -------- | ---------------------- | ----------------- | ----- |
| **keys** | **List&lt;String&gt;** | Keys to split on. |
| **tags** | **List&lt;String&gt;** | Tags to split on. |
