- **Type**: API key
- **API key parameter name**: DD-API-KEY
- **Location**: HTTP header

### apiKeyAuthQuery


- **Type**: API key
- **API key parameter name**: api_key
- **Location**: URL query string

### appKeyAuth


- **Type**: API key
- **API key parameter name**: DD-APPLICATION-KEY
- **Location**: HTTP header

### appKeyAuthQuery


- **Type**: API key
- **API key parameter name**: application_key
- **Location**: URL query string
